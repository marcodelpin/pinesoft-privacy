주요 기능

🎸

다중 음 감지

6개 줄을 동시에 튕기고 한 화면에서 각 줄의 튜닝 상태를 확인하세요. 줄을 한 번에 하나씩 튕길 필요가 없습니다.

🎶

실시간 음높이 감지

SwiftF0 + YIN 시간 영역 자기상관 알고리즘이 장치에서 완전히 실행됩니다. 서브센트 정밀도, 낮은 지연.

⚙

9가지 튜닝 프리셋

스탠다드, Drop D, Open G, Open D, DADGAD, 반음 내림, 온음 내림 등. 커스텀 튜닝도 지원.

📱

완전 오프라인

계정 없음, 네트워크 호출 없음. 오디오는 장치를 절대 떠나지 않습니다 — 알고리즘은 로컬에서 실행됩니다.

🌐

8개 언어

영어, 이탈리아어, 독일어, 프랑스어, 스페인어, 포르투갈어, 일본어, 한국어로 완전 지원.

📈

튜닝 기록

튜닝 횟수와 각 세션의 목표 근접도를 추적하세요. 장치에 로컬로 저장됩니다.

당신의 개인정보

PolyTuner는 마이크 오디오를 장치에서 완전히 처리합니다. 오디오는 절대 기록, 저장 또는 전송되지 않습니다. 익명 튜닝 기록만 장치에 남아 있습니다.
